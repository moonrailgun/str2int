# str2int

Encode string to int

use for select somthing random with string

## Usage

```javascript
var str2int = require('str2int');

str2int('a'); // => 97
str2int('A'); // => 65
str2int('Hello World'); // => 1052
str2int('World Hello'); // => 1052
str2int(''); // => 0
str2int('你好中国'); // => 85511
```
